
# Projeto - AstroMatch 

## Stack
- Esse é um projeto de Frontend Web feito utilizando ReactJS, Styled-Components,
React-Icons e Axios; 
e como gerenciador de pacotes do NodeJS o npm.

- A divisão dos arquivos foi feita da seguinte forma:
Os componentes foram acomodados na pasta 'COMPONENTS' na raiz do projeto.
Ao ingressar na pasta 'COMPONENTS' podemos visualizar a separação dos itens
Dentro da pasta 'ACTIONS' temos o componente que faz a ação, os botões 'Like/Dislike'
e toda sua lógica que inclui consumo de API externa com AXIOS. 'CARDPERSON' nos traz
as imagens dos perfis, novamente com consumo externo de API. 'HEADER' é um componente
estático comum para todos os outros componentes. 'MATCHES' componente onde podemos
visualizar os matches obtidos dos perfis onde decidimos dar LIKE. 'UTILITY' podemos
encontrar o componente 'CONECTION.JS' utilitário criado para usar a conexão com a API
externa, de uso comum em diversos componentes, seu intuito é facilitar e manter o 
código o mais limpo possível.


## Sobre
O Astro-Match foi um projeto agradável, a essa altura imaginar estar habilitado para criar
uma solução que se assemelha a diversas já existentes e consagradas no mercado trouxe, de fato, uma alegria, mesmo com todas as limitações tecnológicas e de 'know-how' foi possível apreciar o seu desenvolvimento desde o início.
